## Emphasis

```markdown
_This text will be italic_
_This will also be italic_
**This text will be bold**
**This will also be bold**
_You **can** combine them_
```

## Lists

### Unordered

```markdown
- Item 1
- Item 2
  - Item 2a
  - Item 2b
```

- Item 1
- Item 2
  - Item 2a
  - Item 2b

### Ordered

```markdown
1. Item 1
2. Item 2
3. Item 3

- Item 3a
- Item 3b
```

1. Item 1
2. Item 2
3. Item 3

   - Item 3a
   - Item 3b

### Slash Commands

When in issues and pull requests you can use the following slash commands to perform tasks.

- /code
- /details
- /saved-replies
- /table
- /tasklist
- /template
